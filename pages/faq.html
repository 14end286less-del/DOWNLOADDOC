Browser-native file compression technology inside DownloadDoc

This file compressor uses modern browser technology to shrink files without ever touching a server. When you use the DownloadDoc file compressor on an image, the file compressor resizes the image through the Canvas API, re-encodes it to your chosen format, and saves the result directly back to your device. For PDFs, the file compressor renders each page through PDF.js and rebuilds the document at reduced quality using pdf-lib. For DOCX files, the file compressor re-packages the internal ZIP archive using maximum DEFLATE compression. This layered approach makes every pass through our file compressor efficient and accurate.
